# Archetype Codes

SUBIT‑64 uses a compact, canonical coding system to identify each semantic archetype.  
Every archetype code is a **three‑character trigram** formed by combining:

- **inner valence (tension)** — one‑letter code  
- **outer valence (color)** — two‑letter code  

This trigram is the operational identity of the element and corresponds directly to its position in the 8×8 SUBIT Crystal.

---

## Inner Valence Codes (1‑letter)

S — Source  
V — Vector  
F — Field  
M — Form  
T — Turn  
C — Compression  
D — Disintegration  
N — Null

---

## Outer Valence Codes (2‑letter)

Bl — Blue  
Gr — Green  
Lm — Lime  
Yl — Yellow  
Or — Orange  
Rd — Red  
Bg — Burgundy  
Pu — Purple

---

## Trigram Format

Each archetype code is written as:

**Inner + Outer → e.g., SBl, MOr, DPu**

This format is:

- minimal  
- unambiguous  
- structurally aligned with the SUBIT Crystal  
- directly mappable to the binary coordinate {0,1}⁶

## Structural Identity

Each archetype code corresponds to:

- **one inner tension**  
- **one outer color**  
- **one trigram identity**  
- **one binary address**  
- **one position in the 8×8 crystal**  

Thus:

**Archetype Code = Trigram = Valence × Color = {8 × 8} = 64**
