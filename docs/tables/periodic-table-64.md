# Periodic Table of 64 SUBIT Elements

The periodic table of SUBIT‑64 arranges all semantic elements into an 8×8 crystalline lattice.  
Rows correspond to **inner valences (tensions)**.  
Columns correspond to **outer valences (colors)**.  
Each cell is a unique **trigram element** formed by combining tension + color.

This table is the canonical structural map of the SUBIT Crystal.

# Structural Interpretation

Each element is defined by:

- **inner tension** (row)  
- **outer color** (column)  
- **6‑bit binary address** (universal SUBIT coordinate)  
- **three bigrams** (WHO, WHERE, WHEN)  
- **two trigrams** (inner + outer)

Thus:

**SUBIT = 8 tensions × 8 colors = 64 archetypes = {0,1}⁶**

The periodic table is the visual and structural expression of this identity.
